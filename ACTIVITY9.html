Correct Answers

1. What does HTML stand for? ✅ Hyper Text Markup Language

2. Which of the following is a style sheet language? ✅ CSS

3. What does CSS stand for? ✅ Cascading Style Sheets

4. What is the primary focus of Back-End Development? ✅ Managing databases

5. Which of the following is a core technology for Front-End Development? ✅ CSS

6. What does JavaScript allow developers to do? ✅ Add dynamic elements to web pages

7. Which of the following is NOT a server-side language? ✅ HTML

8. What is the role of APIs in web development? ✅ To enable software communication

9. What does a full-stack developer do? ✅ Handles both front-end and back-end development

10. Which of the following is a benefit of using JavaScript? ✅ It enables animations and interactive features
